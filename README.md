 Memora

Memora is a photo customization web application that allows users to upload their images and order handcrafted prints . Users Upload their  favorite photo and specify  preferences such as number of people, background style, frame type, size, and any personalized message.The finished artwork arrives beautifully packaged in a luxury gift box, at the given address.
Features
 * Upload personal photos
* Select print size
* Choose orientation (landscape or portrait)
* Simple and user-friendly interface
* Handcrafted product workflow
* Deployed and accessible online

 Tech Stack
* HTML
* CSS
* JavaScript

Live Demo
Deployed on Vercel:
memora-pi.vercel.app

How It Works
1. User uploads a photo
2. Fills out a form with preferences (size, orientation)
3. Submits the request
4. The Memora team handcrafts the product
5. Final product is delivered to the customer

 Setup & Installation
1. Clone the repository:`
git clone https://github.com/your-username/memora.git
2. Open the project folder
cd memora
